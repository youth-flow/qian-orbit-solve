# qian-orbit-solve

Numerical astrodynamics project for Qian Xuesen's solar-return trajectory problem. The code studies Earth launch, real Moon gravity-assist geometry, heliocentric transfer, and Earth return using offline JPL Horizons ephemerides.

The repository root is the runnable project directory required by the course rubric: it contains `report.tex`, `Makefile`, `README.md`, `AI-Agent.md`, and `src/`.

## Features

- Patched-conic reproduction of Qian's reference solar-return example.
- Velocity-Verlet N-body integrator with acceleration, step, and propagation interfaces.
- Offline Sun-Earth-Moon validation against JPL Horizons caches using `CENTER='@10'`.
- Real Moon encounter and real Earth return closure audit for 2026 launch dates.
- Machine-readable mission summaries and timestamped trajectory CSV files.
- Optional extensions: 3D inclination analysis, GR correction estimate, Lambert solver, multi-flyby exploration, Tkinter demo, and surrogate-assisted screening.

## Requirements

- Python 3.10+
- `numpy`, `matplotlib`, `imageio`, `imageio-ffmpeg`
- XeLaTeX for compiling the Chinese report
- `make`

Install Python dependencies:

```powershell
python -m pip install -r requirements.txt
```

## Reproduce

Run the full workflow from the repository root:

```powershell
make all
```

This command:

1. reads the committed offline Horizons caches in `data/`;
2. runs `src/run_all.py`;
3. writes JSON/CSV/PNG/MP4 outputs under `data/generated/`;
4. compiles `report.tex` into `report.pdf`.

Useful partial commands:

```powershell
make data
make pdf
make check
make clean
python src/self_check.py
python src/interactive_demo.py --self-test
```

On Windows systems without GNU Make, use the equivalent commands:

```powershell
python src/run_all.py
xelatex -interaction=nonstopmode -halt-on-error -file-line-error report.tex
xelatex -interaction=nonstopmode -halt-on-error -file-line-error report.tex
python src/self_check.py
```

## Main Outputs

After `make all`, the most important machine-readable files are:

- `data/generated/results.json`
- `data/generated/grading_check.json`
- `data/generated/mission_summary.json`
- `data/generated/today_solution.json`
- `data/generated/scan_daily_best.csv`
- `data/generated/mission_trajectory_day223.csv`
- `data/generated/today_trajectory_day169.csv`

These files contain the launch date, Moon encounter time, Earth return time, lunar periapsis distance, solar perihelion distance, launch/Moon residual/reentry delta-v terms, total delta-v, savings ratio, and timestamped trajectory states.

## Public Release Notes

This public repository intentionally excludes local secrets and large copyrighted reference PDFs. The default build uses offline caches and does not require `JPL_API` or `JPL_TOKEN`. If online Horizons proxy access is needed, copy `JPL_API.example.env` locally and fill in private values outside git.

See `docs/public-release-notes.md` for the full sanitization note.
