# Public Release Notes

This public repository is a sanitized version of the course project.

The following local files are intentionally excluded:

- `JPL_API.env`, because it contains a private Horizons proxy token.
- `PROJECT_SPEC.tex` and `PROJECT_SPEC.pdf`, because the local copy contains course proxy credentials.
- `星际航行概论-钱学森-2008年版.pdf` and `行星航行概论-钱学森-1963年版.pdf`, because they are large copyrighted reference books.
- `data/generated/`, because it is generated by `make all`.

The default build uses committed offline JPL Horizons caches and does not require an online token. If online proxy access is needed, copy `JPL_API.example.env` locally, fill in the real values, and load those variables before calling `src/jpl_access.py`.
